# refolongs 📚

리서치 & 큐레이션 레포. AI 강의 트렌드, AX 도입 사례, 도구, 브랜딩 영감 등 참고 자료를 모아둡니다.

## 폴더 구조

| 폴더 | 내용 |
|------|------|
| `ai-education/` | AI 강의 트렌드, 타 강사 커리큘럼, 교육 자료 |
| `ax-cases/` | 기업 AX 도입 사례, 성공/실패 레퍼런스 |
| `tools/` | AI 도구 리서치, 비교, 사용법 메모 |
| `inspiration/` | 브랜딩, 포트폴리오, 강사 포지셔닝 참고 |

## 사용법

- 폴더별 `README.md`에 링크 + 한 줄 메모로 정리
- 파일명은 `YYYY-MM-주제.md` 형식 권장
- 나중에 찾기 쉽게 태그 달기: `#강의자료` `#사례` `#도구`
